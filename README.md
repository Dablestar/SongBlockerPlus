## (EN) Chrome Extension - Song Blocker
### https://www.youtube.com/watch?v=c4KNd0Yv6d0&ab_channel=FriedFish

## 1. Reason for Development
 To make a better experience of doomscrolling by skipping hated meme songs(Skibidi, Japanese Anime Song, etc..) keep appearing in Youtube Shorts.

## 2. Environment
 - Javascript
 - Chrome Extension API

## 3. How does it work?
 1) Extract source music URL from Shorts
 2) Add on Ban List(localStorage)
 3) Check source music before shots play. If the source is included in localStorage, call skip functionality using Youtube iFrame API

## 4. How to use(After it released)?
 1) Download the zip file and extract it (path doesn't matter)
 2) Chrome Options > Extensions > Manage Extensions > Load Unpacked
 3) Select the folder you extracted at 1)
